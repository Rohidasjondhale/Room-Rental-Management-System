# Room Rental Management System

## Project Overview
A full-stack web application where Owners can list rental properties and Tenants can browse, book, and manage room rentals online.

## Features
- User Signup / Login (JWT Authentication)
- Owner Dashboard
- Add / Edit / Delete Property
- Tenant Dashboard
- Browse Available Properties
- Booking Management
- Payment Integration (Cashfree)
- Forgot Password (Brevo Email)
- Owner Contact Details
- Booking Cancellation
- Responsive UI

## Tech Stack
### Frontend
- HTML
- CSS
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MySQL
- Sequelize ORM

### Authentication
- JWT
- bcryptjs

### Third Party Services

- Brevo Email Service
